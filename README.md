## Getting Started

First, run the development server:

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

please should note that,le if you test api request, pase run following command:

```bash
cd server

pnpm start
```
